진단평가 & 수준별 콘텐츠

수업 시작 전 간단한 진단평가로 학생 이해도를
파악하고, AI가 분석하여 각 학생에게 필요한
보충, 심화 과제를 자동으로 추천합니다.

교과서 예시: '알고리즘' 단원

AI가 '가위바위보' 프로그램 코드 작성 능력을 진단하고, 변수 개념이 부족한 학생에겐 관련 영상을, 조건문 활용이 미숙한 학생에겐 추가 예제를 제공합니다.

자동 피드백 & 오답 노트

학생이 코드를 제출하면 AI가 즉시 채점하고
실시간 피드백을 제공합니다. 문법 오류, 논리적 실수 등을 구체적으로 짚어줍니다.

교과서 예시: '알고리즘' 단원

'가위바위보' 코드에서 잘못된 비교 연산자를 사용했을 때, AI가 즉시 "=="와 "="의 차이점을 설명하는 도움말 링크를 보여줍니다.

게임형 학습 콘텐츠

드래그 앤 드롭으로 코드 블록을 맞추거나, 순서도를
완성하는 등 게임처럼 즐기는 학습 활동으로 학생들의 몰입감을 높입니다.

교과서 예시: '알고리즘' 단원

'가위바위보' 로직을 순서도 블록을 드래그하여 완성하는 게임을 통해, 학생들은 알고리즘의 절차적 사고를 재미있게 학습합니다.

함께 코딩 & 모둠 활동 지원

여러 학생이 하나의 코드 파일을 동시에 편집하며
협업하고, 교사는 모둠별 프로젝트 진행 상황을
실시간으로 확인할 수 있습니다.

교과서 예시: '알고리즘' 단원

모둠원들이 '함께 코딩' 모드로 '가위바위보' 게임을 확장하여 '묵찌빠' 기능을 추가하는 프로젝트를 진행하고, AI는 각자의 코드 기여도를 자동으로 기록합니다.

인솔 모드: 교사 주도 실시간 활동

교사가 전체 학생의 화면을 제어하며 수업을
이끌어갑니다. 복잡한 개념을 설명하거나 모두의
학습 흐름을 맞춰야 할 때 효과적입니다.

교과서 예시: '함수' 단원

교사가 '인솔 모드'로 'return' 개념을 설명합니다. 모든 학생 화면에 동일한 코드가 나타나고, 교사가 실시간으로 코드를 수정하며 설명하면 모든 학생 화면에 동기화되어 나타납니다.

실습 코칭 모드: AI 페어 프로그래머

'함께 코딩' 중 막히는 부분이 생겼을 때, AI 코치가
정답 대신 문제 해결에 필요한 힌트를 주어
자기주도적 학습을 돕습니다.

교과서 예시: 'API 활용' 프로젝트

API 데이터 처리에 어려움을 겪는 모둠이 '코칭 모드'를 켭니다. AI는 데이터 구조를 확인하는 방법을 제안하고, 특정 정보에 접근하는 방법에 대한 힌트를 주어 학생들이 스스로 해결하도록 유도합니다.
